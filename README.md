# 🚀 DigiTools

## 📌 Project Name

**DigiTools – AI & Digital Tools Marketplace**

---

## 📝 Description

**DigiTools** is a modern web interface for an **AI and digital productivity tools marketplace** where users can explore and purchase various tools such as AI writing assistants, resume builders, automation tools, design templates, and stock assets.

The platform presents different tools in a clean **card-based pricing layout**, allowing users to easily compare features, pricing models (monthly or one-time), and special tags like **Popular**, **Best Seller**, and **New**.

The project focuses on building reusable UI components, structured data rendering, and responsive design using modern frontend technologies.

---

## ⚙️ Technologies Used

- React.js
- JavaScript (ES6+)
- Tailwind CSS
- React Icons
- React Toastify
- Vite
- Vercel (Deployment)

---

## ✨ Key Features

### 💳 Digital Tools Marketplace

Users can browse multiple AI and productivity tools with detailed pricing, descriptions, and included features.

### 🧩 Dynamic Card Rendering

All tools are rendered dynamically from structured **JSON data**, making the interface scalable and easy to update.

### 🎨 Modern Responsive UI

Built with **Tailwind CSS**, the design provides a clean, responsive layout that works across desktop, tablet, and mobile devices.
